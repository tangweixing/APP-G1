# G1 键盘遥控器操作手册

## 启动方式

```bash
cd ~/tang/WK/unitree_sdk2_python/example/g1/high_level
python3.9 g1_keyboard_teleop.py eth0
```

## 按键总览

### 运动控制（始终可用）

| 按键 | 功能 | 说明 |
|------|------|------|
| W | 前进 | 按住前进，松开自动减速停止 |
| S | 后退 | 按住后退，松开自动减速停止 |
| A | 左移 | 机器人横向左移 |
| D | 右移 | 机器人横向右移 |
| Q | 左转 | 原地左转 |
| E | 右转 | 原地右转 |
| Space | 急停 | 立即停止所有运动 |
| , | 降档 | 低速→中速→高速，降一档 |
| . | 升档 | 低速→中速→高速，升一档 |
| R | 速度归零 | 目标速度清零，平滑停止 |

### 档位速度参数

| 档位 | 前进/后退 | 左移/右移 | 左转/右转 |
|------|----------|----------|----------|
| 1 低速 | 0.3 m/s | 0.2 m/s | 0.4 rad/s |
| 2 中速 | 0.6 m/s | 0.3 m/s | 0.8 rad/s |
| 3 高速 | 1.0 m/s | 0.5 m/s | 1.2 rad/s |

### 模式切换

| 按键 | 功能 | 说明 |
|------|------|------|
| F | 切换 Arm/FSM 模式 | 在两种数字键模式间切换，状态栏显示 [Arm] 或 [FSM] |

### Arm 模式（默认，数字键控制机械臂）

| 按键 | 动作 | 自动复位 |
|------|------|---------|
| 0 | 释放手臂 | 否 |
| 1 | 握手 | 是（2秒后复位） |
| 2 | 举手你好 | 是 |
| 3 | 拥抱 | 是 |
| 4 | 拜拜挥手 | 否 |
| 5 | 鼓掌 | 否 |
| 6 | 低位挥手 | 否 |
| 7 | 左飞吻 | 否 |
| 8 | 比心 | 是 |
| 9 | 右比心 | 是 |
| Z | 双手举起 | 是 |
| X | 迪迦光线 | 是 |
| C | 右手举起 | 是 |
| V | 拒绝 | 是 |
| B | 右飞吻 | 否 |
| N | 双手飞吻 | 否 |

### FSM 模式（按F切换后，数字键切换运动状态）

| 按键 | FSM ID | 状态 | 说明 |
|------|--------|------|------|
| 0 | 0 | 零力矩 | 关节无力矩，机器人瘫软 |
| 1 | 1 | 阻尼模式 | 关节有阻尼，机器人缓慢下沉 |
| 3 | 3 | 坐下 | 机器人坐下 |
| 4 | 4 | 预备模式 | 机器人进入预备姿态 |
| 5 | 501 | 常规走路 | 切换到走路模式 |
| 6 | 200 | 启动运动 | 启动运动控制 |
| 7 | 702 | 躺→站 | 从躺姿站起（需面朝上） |
| 8 | 706 | 蹲↔站 | 蹲下/站起切换 |
| 9 | 802 | 走跑模式 | 切换到走跑模式 |

### 退出

| 按键 | 功能 |
|------|------|
| Ctrl+C | 退出程序，自动停止运动并释放手臂 |

## 状态栏说明

运行时底部实时显示状态信息：

```
  [Arm] 档位:1(低速) | 目标:(+0.30,+0.00,+0.00) | 实际:(+0.28,+0.01,+0.00) | 🤖空闲
```

| 字段 | 含义 |
|------|------|
| [Arm] / [FSM] | 当前数字键模式 |
| 档位 | 当前速度档位 |
| 目标 | 目标速度 (vx, vy, wz) |
| 实际 | 机器人实际速度 (vx, vy, wz) |
| 🤖空闲 / 🤖执行中 | 机械臂动作执行状态 |

## 典型操作流程

### 1. 基本行走

```
启动 → 按 W 前进 → 松开自动停止 → 按 . 升档 → 按 W 加速前进
```

### 2. 切换走路/走跑模式

```
按 F → 进入 FSM 模式 → 按 5 切换走路 → 按 9 切换走跑 → 按 F 回到 Arm 模式
```

### 3. 边走边做动作

```
按 W 前进 → 按 1 握手（Arm模式下）→ 机器人边走边握手 → 2秒后自动复位
```

### 4. 紧急停止

```
任何时候按 Space → 机器人立即停止
```

## 安全注意事项

1. 首次使用建议从 **1档（低速）** 开始
2. 切换 FSM 模式前确保机器人处于稳定状态
3. FSM ID 0（零力矩）会导致机器人瘫软，确保有人扶住
4. 退出程序时会自动发送 StopMove + 释放手臂，但仍建议先手动急停再退出
